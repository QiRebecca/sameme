You are designing a playful but credible AI demo UI.

Design rules:
- Use a warm dark theme by default: ink, plum, amber, soft cyan.
- Use glass cards, soft borders, gradient glows, and subtle grain.
- Make the app feel like "人类也不过如此博物馆" / "historical emotional mirror".
- Use motion only for transform/opacity/layout. Avoid expensive animation.
- Every screen must work on mobile.
- Avoid tiny low-contrast text.
- UI must clearly separate:
  [史实] factual basis
  [类比] modern analogy
  [演绎] persona dramatization
- Candidate cards should feel collectible, like summoning cards.
- Group chat should feel alive: staggered message bubbles, avatar badges, tiny role labels.
- Final CTA must be a tiny action, not generic motivation.

Preferred components:
- Button, Card, Badge, Tabs, Tooltip, Sheet/Drawer, ScrollArea, Textarea.
- Animated background: soft radial gradients, grid dots, floating orbs.
- Persona cards: moving border, hover lift, source confidence badge.
- Agent trace: vertical or horizontal timeline with checkmarks and loading states.
