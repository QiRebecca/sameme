# 俺也一样 / SameMe

## 一句话

**俺也一样** 是一个“历史名人 Agent 陪伴与成长”产品：当用户觉得“怎么只有我会这样”的时候，系统会召唤经历过相似人生困境的历史人物进入群聊，用他们的处境、性格、幽默和选择，陪用户把焦虑说清楚，把孤独感放下来，再把今天写成一页自己的成长小史。

## Why Now

现代人并不缺信息，缺的是一种更温柔、更有纵深的陪伴。

很多人遇到拖延、职场失意、创作停滞、自我怀疑、意义感下降时，第一反应不是求助，而是觉得“是不是只有我这么差”。但人类历史本身就是一座巨大的情绪样本库：胡适会拖延和自我观察，苏轼经历贬谪后重建生活，王阳明把“知道”推向“行动”，庄子帮人松开“必须有用”的绳子。

**俺也一样** 把这些历史经验做成可对话的 Agent，不是讲大道理，而是让用户真的感觉：原来这些困境不是我一个人的缺陷，而是人类反复遇到的处境。

## Product Vision

我们希望做一款每个人都会需要一点的成长型陪伴产品。

它不是传统心理咨询替代品，也不是简单的名人语录生成器，而是一个“历史情绪镜像系统”：

- 用户说出当下真实烦恼。
- 系统匹配有相似人生阶段的历史人物。
- 名人 Agent 进入群聊，用拟态人格和真实生平底座进行陪伴、追问、吐槽和建议。
- 史官 Agent 把用户的分享、困惑和当日回应整理成“人生小记”。
- 每段小记配插图、日期和评论，形成用户自己的成长档案。

**以史为鉴，不只是知兴替，也可以知自己。**

## Project Idea

**俺也一样** 的核心想法很简单：把“我是不是只有自己这样”的孤独感，变成“原来很多厉害的人也曾经这样”的连接感。

传统陪伴产品通常从用户当下情绪出发，给安慰、建议或行动清单。但很多时候，用户真正需要的不是立刻被纠正，而是先被放回一个更大的叙事里：拖延不是一种现代人才有的失败，仕途受挫不是人生终局，创作停滞也不是才华消失。历史上那些被我们称为“先贤”“名人”“伟人”的人，也曾经被现实拦住、被评价刺伤、被惰性拖住、被前途吓住。

所以这个产品不把历史人物做成冷冰冰的知识卡片，而是把他们做成可进入群聊的 Agent。用户可以像和朋友聊天一样问苏轼：“你被贬的时候真的不难受吗？”也可以问胡适：“你拖延之后会不会自责？”系统会把用户的处境映射到历史人物的相似人生阶段，让用户从对方的经历、选择和后续人生里获得一种更有重量的陪伴。

这也是“俺也一样”这个名字的来源：当用户说出自己的狼狈时，历史人物不是站在高处教育他，而是坐到旁边说一句，俺也一样。

## Innovation

### 1. 从“情绪识别”升级到“人生处境匹配”

多数情绪产品会把用户输入归类为焦虑、拖延、低落或压力大。**俺也一样** 进一步追问：这种情绪更像历史上哪一种人生处境？

例如“我好懒真的不想干了”不只是拖延，它可能同时包含低能量、自责、启动困难和对失败的预期。系统会把这些信号映射到胡适的自我观察、苏轼的低谷修复、王阳明的知行张力、庄子的反内耗视角。匹配的不是标签，而是人与人的相似处境。

### 2. Persona Memory 让名人 Agent 有来处

产品的关键不是让模型模仿名人说话，而是让每个 Agent 带着自己的“记忆底座”进入对话。

每位历史人物都可以拥有：

- 生平阶段：什么时候顺利，什么时候受挫，什么时候转向。
- 作品与表达：诗文、日记、书信、思想关键词。
- 典型困境：被贬、拖延、失败、孤独、怀才不遇、意义危机。
- 应对方式：如何重新组织生活，如何行动，如何自我解释。
- 语气人格：幽默、自嘲、豪放、克制、犀利或松弛。
- 事实边界：哪些可以演绎，哪些不能伪造成原话。

这使得 Agent 的回答不只是“AI 安慰话术”，而是来自一个有历史厚度的人格上下文。

### 3. 多 Agent 群聊比单 Agent 更接近真实陪伴

人的困惑往往不是一个答案能解决的。苏轼可能先安慰，王阳明可能催行动，庄子可能拆掉内耗，胡适可能拿自己的拖延来自嘲。多个 Agent 在同一群聊里形成不同角度，用户可以选择听谁的、@谁追问、拍一拍谁，甚至看他们互相接话和拆台。

这种设计把 AI companion 从“一对一答案机”变成一个有关系、有节奏、有角色张力的陪伴现场。

### 4. 史官 Agent 把聊天变成长期成长资产

很多陪伴产品的问题是聊完就消失了。**俺也一样** 会把用户当天的烦恼、对话、转念和行动写成一页“人生小记”，配上插图、日期和名人评论。

这让用户不是在反复倾倒情绪，而是在积累自己的成长史。几周后回看，用户能看到自己曾经怎样卡住、谁陪他说过什么、自己后来怎样动了一点点。

### 5. 历史内容产品的新形态

历史通常被做成课程、文章、短视频或百科。**俺也一样** 把历史变成一个可交互、可陪伴、可长期记忆的 Agent 系统。

用户不是被动学习“苏轼是谁”，而是在自己职场失意时问苏轼“你当时怎么办”；不是背诵“知行合一”，而是在启动不了时被王阳明拉回具体行动。历史知识因此从内容消费变成现实生活中的心理支点。

## Demo Experience

打开首页后，用户会进入一个像即时通讯软件一样的 Agent 群聊空间。用户先输入一句真实烦恼，系统会在后台完成情绪理解、相似人生检索、人物上下文组装和群聊召唤：

- 用户输入：“我好懒真的不想干了”
- 系统理解这是低能量、拖延、自责和启动困难混合在一起的状态。
- RAG 检索历史人物的生平阶段、日记/书信/作品线索和关键人生事件。
- Persona Matcher 找到经历过相似困境的人物，比如胡适、苏轼、王阳明、庄子。
- 用户从人物卡片中选择想拉进群聊的对象。
- 被选中的历史人物 Agent 带着自己的语气、记忆和处境进入聊天。

Demo 展示能力包括：

- 历史名人 Agent 群聊
- `@苏轼` 这类定向提问
- 点击头像“拍一拍”
- 名人 Agent 主动判断是否发言、是否接话、是否互相补充
- Persona Memory 让回答围绕人物真实处境展开，而不是泛泛安慰
- 史官 Agent 自动生成小传
- 日历跳转历史小传
- 懒猫水墨插图作为小传视觉资产

## Core User Value

### 1. 让用户知道：不是只有你这样

当用户拖延、自责、焦虑、失落时，系统不是立刻给任务清单，而是先帮用户找到历史上的相似处境。

“你不是坏掉了。历史上很多聪明、伟大、有创造力的人，也有过类似低谷。”

这种共鸣本身就是强陪伴。

### 2. 让历史人物从知识变成可陪伴的 Agent

用户不再只是阅读苏轼、胡适、王阳明，而是可以问：

- “你被贬的时候，真的不难受吗？”
- “你拖延之后会不会很自责？”
- “知道该做但完全启动不了怎么办？”

Agent 会基于人物的生平阶段、思想气质和语言风格进行拟态回应。重点不是复述知识，而是把历史经验转化成对当下困境有用的陪伴。

### 3. 把情绪出口变成成长记录

用户在群聊里的困惑、转念、行动，会被史官 Agent 记录成每日小传。

这不是普通日记，而是一种带有历史叙事感的成长档案：

- 今天发生了什么
- 用户当时卡在哪里
- 哪些名人 Agent 参与了回应
- 他们如何类比自己的经历
- 用户从中获得了什么新的理解

长期来看，这会形成用户自己的“人生小史”：既可回看经验，也能记录成长。

## Product Modules

### Historical Persona Council

多名历史人物 Agent 组成群聊，不是每轮都机械发言，而是根据用户问题判断谁更适合回应。

示例角色：

- 胡适：自责型摸鱼代表，适合拖延、自我观察、重新开始。
- 苏轼：低谷生活修复师，适合职场受挫、失意、生活感重建。
- 王阳明：知行合一行动启动器，适合知道该做但启动不了。
- 庄子：反内耗观察员，适合意义危机、外界评价、自我松绑。

### Agentic Conversation

群聊像微信，而不是问答机器人。

支持：

- 用户自由输入
- `@某个名人` 定向提问
- 点击头像“拍一拍”
- Agent 之间互相接话、拆台、补充
- 名人可选择沉默，不强制每轮全部回复

### Biography Agent

史官 Agent 会把当天聊天写成短小的人生小传，风格介于史书、日记和轻喜剧之间。

例如：

> 某日，小友言“我好懒真的不想干了”，遂召胡适、苏轼、王阳明、庄子入群。胡适以日记式自嘲接住其羞意，苏轼举黄州低谷，王阳明催其知行相接，庄子按住内耗之鞭。

### Personal Growth Calendar

小传按日期归档，用户可以通过日历查看哪天有记录、哪天没有记录，并快速跳转。

这让产品不只是一次聊天，而是持续陪伴用户成长的容器。

## Market Opportunity

情绪陪伴、AI companion、心理健康、个人成长、知识型内容产品正在快速融合。大量用户需要的不是严肃治疗，也不是泛泛鸡汤，而是一种低压力、可持续、有文化质感的陪伴方式。

**俺也一样** 的差异点在于：

- 用历史人物降低孤独感，而不是只用虚构角色。
- 用生平阶段匹配用户处境，而不是只做情绪标签。
- 用群聊形态降低表达门槛。
- 用史官小传沉淀长期成长资产。
- 兼具陪伴、知识、内容生成和个人记录四种价值。

## Business Potential

可能的商业化方向：

- 个人订阅：更多历史人物、更长记忆、更完整成长档案。
- 青少年/大学生成长陪伴：学业焦虑、拖延、自我认同。
- 职场人情绪支持：被评价、晋升受挫、职业迷茫。
- 内容社区：用户可选择分享自己的“小传”或匿名故事。
- IP 化：历史人物 Agent、主题卡组、节气小传、年度成长史。
- B2B 教育与文化产品：历史学习、作文素材、通识教育陪伴。

## Safety and Historical Grounding

项目明确区分事实、类比和演绎：

- 名人发言是拟态演绎，不代表历史人物真实原话。
- 历史材料作为人格和处境底座，不伪造直接引语。
- 胡适相关内容会说明：日记中确有打牌、学习、上课、自我约束等记录；网传“胡适之啊胡适之”式段子属于风格化复述/二创，不等同于原文。
- 高风险自伤表达不进入搞笑群聊，而应进入安全支持模式。

## Current Demo

当前版本展示的是一条完整的 Agent 产品链路：

- **情绪理解 Agent**：把用户的自然语言烦恼解析成情绪、行为和当下需求。
- **历史检索 Agent**：从人物资料、作品线索、传记事件和历史语境中寻找相似处境。
- **Persona Memory Agent**：为每位历史人物组装可对话的上下文，包括人生阶段、典型选择、性格张力、表达习惯和需要避免的事实误读。
- **群聊调度 Agent**：决定谁适合加入群聊、谁应该先回应、谁可以沉默、谁适合补刀或安慰。
- **历史人物 Agent**：以拟态人格参与对话，讲自己的相似经历、当时如何想、如何选择、后来结果如何。
- **史官 Agent**：根据当天聊天生成一页“人生小记”，附上插图、日期和名人评论，形成用户自己的成长档案。

这不是一个简单的聊天机器人，而是一个由多 Agent 协作完成的“历史经验陪伴系统”：先理解用户，再找到历史镜像，然后让这些人物像朋友一样坐进群聊，最后把用户今天的经历写进自己的小史。
