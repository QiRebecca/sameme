# AGENTS.md

## Project
This is a hackathon demo for "先贤捞捞 / SageMatch", a playful historical-persona agent that maps a user's emotional phrase to analogous historical life events and opens 1v1 or multi-agent chats.

## Working agreements
- Prefer TypeScript strictness.
- Keep the demo runnable without external API keys.
- If adding AI calls, always implement deterministic fallback.
- Separate factual evidence, modern analogy, and dramatized dialogue.
- Never present generated persona dialogue as direct historical quotes.
- Run lint/build after modifying the app if scripts exist.
- Prioritize a beautiful, polished demo over production complexity.

## UX principles
- Cute, warm, slightly witty, non-judgmental.
- No heavy lectures.
- End each conversation with a tiny action under 10 minutes.
- Use Chinese as primary product language, with clean English variable names.
